# 08. Pagination

Grids use a default page size, but that can be changed in the `<navigation><pager/></navigation>` section of a grid XML file.


```markup
<navigation>
    <pager>
        <defaultPageSize>5</defaultPageSize>
        <pageSizes>2,5,10</pageSizes>
    </pager>
</navigation>
```
